# Student Management System - CRUD Application

A robust CRUD (Create, Read, Update, Delete) application built using the **.NET Core** framework. This project manages student records and demonstrates backend development best practices.

## 🚀 Features
* **Create:** Add new student records to the database.
* **Read:** View a list of all registered students.
* **Update:** Modify existing student information.
* **Delete:** Remove student records securely.

## 🛠️ Tech Stack
* **Language:** C#
* **Framework:** ASP.NET Core MVC / Web API
* **Database:** Microsoft SQL Server
* **ORM:** Entity Framework Core
* **Frontend:** Razor Views, HTML, CSS, Bootstrap

## 📂 Project Structure
* **Controllers:** Handles the logic and communication between the UI and Database.
* **Models:** Defines the Student entity and data structure.
* **Data:** Contains the DbContext for Entity Framework.
* **Migrations:** Database schema version control.

## ⚙️ How to Run
1. Clone the repository.
2. Update the connection string in `appsettings.json` to point to your local SQL Server.
3. Run `Update-Database` in the Package Manager Console.
4. Press `F5` to run the application in Visual Studio.
